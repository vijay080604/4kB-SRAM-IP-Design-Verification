# Project Overview

This repository documents the design and verification journey of a 1024×32 SRAM (32 Kbits / 4 KB) using OpenRAM and the Google SkyWater SKY130 Process Design Kit (PDK) as part of the SRAM IP Design & Verification Internship.

The reference design targets:

* Memory Size: 1024 × 32 SRAM
* Capacity: 32 Kbits (4 KB)
* Technology: Google SkyWater SKY130 PDK
* Operating Voltage: 1.8 V
* Target Access Time: < 2.5 ns
* Memory Compiler: OpenRAM

The project investigates how OpenRAM utilizes the SKY130 PDK to generate and characterize SRAM memories while also exploring the underlying circuit building blocks responsible for memory operation.

The work focuses on:

* OpenRAM architecture exploration
* SKY130 technology investigation
* SRAM architecture understanding
* 6T SRAM bitcell analysis
* Read and write operation analysis
* Peripheral circuit analysis
* SPICE-based simulation and verification
* AI-assisted engineering workflows
